Emacs 实用技巧

发表于 2015-03-25   |   分类于 软件技巧

快捷方式

  • C-x b：切换 buffer
  • C-x right：向右历遍 buffer
  • C-x left：向左历遍 buffer
  • C-x k：关闭 buffer
  • C-x 0：关闭当前活动窗口
  • C-x o：将下一个窗口设为活动窗口
  • C-x C-w：将文件另存为
  • C-_：撤消
  • M-%：询问替代，space为替代，n为跳过，!为全部替代
